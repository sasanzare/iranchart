## Overview

IranChart is a cryptocurrency education project designed to improve public understanding of digital currencies. The existing README describes educational outreach, concept simplification, interactive learning tools, questionnaires, and article-based learning.

## Highlights

- Cryptocurrency education platform
- Frontend and backend directories
- Interactive learning and questionnaire concept
- Article-driven knowledge base positioning
- Broad web stack including CSS, JavaScript, SCSS, Python, and HTML
